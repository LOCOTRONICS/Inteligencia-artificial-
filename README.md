# Inteligencia-artificial-
En este repositorio tenemos el codigo para entrenar un modelo de IA y ejecutarlo en tiempo real 

Primero se debe entrenar la red neuronal con el archivo de entrenamiento, al descargarse el archivo .pt debe tenerse la ruta donde se ha guardado.
Despues se debe copiar la ruta del archivo .pt y pegar en el codigo de implementacion en tiempo real.
Finalmente se ejecuta el codigo de implementacion.

en la linea de codigo de cv2.videocapture(0), se puede usar el 0 para la webcam de la computadora o 1 para una webcam externa
